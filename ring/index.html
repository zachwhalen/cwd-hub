To add your site to this ring, please submit your site using this form, and then copy and paste the following code somewhere in the body of your webpage. 

<div id="garden">
<script type="text/javascript" src="https://garden.dgst101.net/onionring-variables.js"></script>
<script type="text/javascript" src="https://garden.dgst101.net/onionring-widget.js"></script>
</div>

Then, add this line of code to the head of your webpage. 

<link rel="stylesheet" href="https://garden.dgst101.net/onionring.css">

New sites must be added manually, so please be patient if you don't see your website in the ring right away.
